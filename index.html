My personal coding cheatsheet

Table Tags

Tag Name Description
<table> Table Defines html table
<thead> Table Head Defines set of rows for column headers
<tbody> Table Body Defines set of rows containing table data
<tr> Table Row Table row containter
<td> Table Data Table data containter
<tfoot> Table Foot Defines set of rows containing table footer

Code Example

<table>
    <thead>
        <tr>
            <td>Name</td>
            <td>Age</td>
            <td>Hight</td>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Marc</td>
            <td>21</td>
            <td>174cm</td>
        </tr>
        <tr>
            <td>Ula</td>
            <td>38</td>
            <td>156cm</td>
        </tr>
    </tbody>
</table>
Name Age Height
Marc 21 174cm
Ula 38 156cm

Position Values

Value FoD Stacks TRBL Description
Static Yes No No Default element position
Relative Yes Yes Yes Enables use of TRBL offsets and z-index
Absolute No Yes Yes Offsets element relative to closest ancestor
Fixed No Always Yes Offsets element relative to the initial containing block established by the viewport
Sticky Yes Always Yes Scrolling effect. Offsets relative to its nearest scrolling ancestor and containing block

FoD: stays in flow of document

Stacks: enables z-index

TRBL: enables top, right, bottom, left properties
